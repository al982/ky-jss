Array Object

Array adalah objek bawaan untuk menyimpan koleksi elemen yang dapat diakses berdasarkan indeks. Ini mendukung metode seperti push, pop, map, dll. Saya pahami ini sebagai struktur data dinamis yang fleksibel untuk manipulasi data.
